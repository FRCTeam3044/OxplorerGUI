# OxplorerGUI - WIP

A frontend for oxplorer to easily visualize and edit paths in real time as well as visiualizing and editing autos.

## Development

1. Clone the repository
2. Install dependencies

```bash
yarn
```

3. Start the development server

```bash
yarn start
```

4. Build the app

```bash
yarn make
```
